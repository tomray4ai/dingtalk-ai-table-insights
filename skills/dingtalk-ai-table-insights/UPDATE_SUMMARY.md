## 🔧 技术细节

### 为什么需要分页？

**问题**:
- 单次读取大数据表（>100 条）时，JSON 响应超过 60KB
- 命令行输出可能被截断
- JSON 解析失败

**解决方案**:
```python
# 分页读取
while True:
    data = read_with_cursor(doc_id, sheet_id, limit=50, cursor=cursor)
    records = data['result']['records']
    cursor = data['result']['cursor']
    all_records.extend(records)
    
    if not cursor:
        break
```

### 临时文件策略

**原因**:
- 直接捕获大输出可能导致内存问题
- 使用文件更稳定

**实现**:
```python
tmp_file = tempfile.mktemp(suffix='.json')
subprocess.run(cmd, shell=True, stdout=open(tmp_file, 'w'))
data = json.load(open(tmp_file))
os.unlink(tmp_file)  # 自动清理
```
